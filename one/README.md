# SWE Java Assignments

A collection of Java programming assignments completed as part of Software Engineering coursework. This repository demonstrates proficiency in Java programming, object-oriented design principles, and software development best practices.

## 📚 About

This repository contains various Java assignments that cover fundamental to advanced concepts in software engineering and Java programming. Each assignment is organized in its own directory with complete source code, documentation, and any necessary resources.

## 🗂️ Repository Structure

```
swe-java-assignments/
├── assignment-1/          # [Assignment Name/Topic]
├── assignment-2/          # [Assignment Name/Topic]
├── assignment-3/          # [Assignment Name/Topic]
└── README.md
```

## 🎯 Topics Covered

The assignments in this repository cover a range of topics including:

- **Object-Oriented Programming (OOP)**: Classes, objects, inheritance, polymorphism, encapsulation, and abstraction
- **Data Structures**: Arrays, ArrayLists, LinkedLists, Stacks, Queues, Trees, and HashMaps
- **Algorithms**: Sorting, searching, recursion, and algorithm analysis
- **Design Patterns**: Common software design patterns and their implementations
- **File I/O**: Reading from and writing to files
- **Exception Handling**: Proper error handling and custom exceptions
- **Testing**: Unit testing with JUnit
- **Software Design**: UML diagrams, class design, and system architecture

## 🛠️ Technologies Used

- **Java**: JDK 8 or higher
- **IDE**: IntelliJ IDEA 
- **Build Tool**: Maven / Intellij
- **Testing**: JUnit
- **Version Control**: Git & GitHub


### Installation

1. Clone the repository:
```bash
git clone https://github.com/mustah21/swe-java-assignments.git
cd swe-java-assignments
```

2. Navigate to a specific assignment directory:
```bash
cd assignment-1
```

3. Compile the Java files:
```bash
javac *.java
```

4. Run the main program:
```bash
java Main
```

